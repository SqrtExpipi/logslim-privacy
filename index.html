LogSlim Privacy Policy

Last updated: 2026-05-24

This privacy policy applies to the Chrome extension LogSlim - Log Compressor for AI / LogSlim - AI向けログ圧縮.

  • Fully local processing
  • No external communication from the extension
  • No log storage
  • No analytics or tracking
  • Clipboard access is used only after explicit user action

Data LogSlim Handles

LogSlim may process log text from files you choose or drag and drop, text read from the clipboard when you click the "Compress clipboard" button, compressed log output generated by LogSlim, and extension settings such as theme, processing options, masking preference, and auto-copy preference.

Data Collection

LogSlim does not collect, transmit, sell, or share your logs, clipboard contents, compressed output, or settings. The extension does not send your data to any server. It does not use analytics, tracking pixels, advertising SDKs, or remote logging.

Log Storage

LogSlim does not store original logs or compressed logs. Settings are stored locally only so the extension can remember your preferences. These settings do not include raw log content.

Clipboard Permissions

LogSlim requests clipboardRead and clipboardWrite permissions.

  • clipboardRead is used only when you click the "Compress clipboard" button. LogSlim reads the current clipboard text and processes it locally.
  • clipboardWrite is used only when you click "Copy to clipboard", or when the user-enabled auto-copy option writes the processed output after compression.

LogSlim does not monitor the clipboard in the background.

File Processing

Files selected or dropped into LogSlim are processed locally in the browser. Files are not uploaded or transmitted.

Simple Masking

LogSlim includes simple masking for common secret-like values such as emails, IP addresses, URLs, bearer tokens, JWTs, API-key-like strings, passwords, secrets, connection strings, GitHub-token-like strings, AWS-access-key-like strings, and Azure DevOps URLs.

This masking is a safety aid. It does not guarantee that all sensitive information will be detected or removed. Always review the output before sending it to an AI tool or any third party.

Third-Party Services

The LogSlim extension does not communicate with third-party services.

This privacy policy page is hosted on GitHub. GitHub may process access information for visitors to this page according to GitHub's own policies. This does not affect the LogSlim extension, which does not contact GitHub.

Changes

This policy may be updated when LogSlim changes how it handles data or permissions. The "Last updated" date will be changed when this policy is updated.

Contact

For privacy questions, please open an issue in this public repository: https://github.com/SqrtExpipi/logslim-privacy/issues

LogSlim プライバシーポリシー

最終更新日: 2026-05-24

このプライバシーポリシーは、Chrome拡張機能 LogSlim - Log Compressor for AI / LogSlim - AI向けログ圧縮 に適用されます。

  • 完全ローカル処理
  • 拡張機能からの外部通信なし
  • ログ保存なし
  • 分析・トラッキングなし
  • クリップボードアクセスは明示的なユーザー操作時のみ

LogSlimが扱うデータ

LogSlimは、ユーザーが選択またはドラッグ＆ドロップしたファイル内のログテキスト、「クリップボードから圧縮」ボタンを押した時に読み取るクリップボード内のテキスト、LogSlimが生成した圧縮後ログ、テーマ・処理設定・マスキング設定・自動コピー設定などの拡張機能設定を、ユーザーのブラウザ内でローカルに処理する場合があります。

データ収集

LogSlimは、ログ、クリップボード内容、圧縮後ログ、設定を収集、送信、販売、共有しません。拡張機能はデータをサーバーへ送信しません。分析ツール、トラッキングピクセル、広告SDK、リモートログ送信も使用しません。

ログ保存

LogSlimは、元ログや圧縮後ログを保存しません。設定値のみ、ユーザー設定を保持するためにローカルに保存します。これらの設定にはログ本文は含まれません。

クリップボード権限

LogSlimは clipboardRead と clipboardWrite 権限を要求します。

  • clipboardRead は、「クリップボードから圧縮」ボタンを押した時だけ使用します。現在のクリップボード内テキストを読み取り、ローカルで処理します。
  • clipboardWrite は、「クリップボードにコピー」ボタンを押した時、またはユーザーが有効化した自動コピー設定により処理完了後の出力を書き込む時だけ使用します。

LogSlimは、バックグラウンドでクリップボードを監視しません。

ファイル処理

LogSlimに選択またはドロップされたファイルは、ブラウザ内でローカル処理されます。ファイルはアップロードまたは送信されません。

簡易マスキング

LogSlimには、メールアドレス、IPアドレス、URL、Bearer token、JWT、API key風の長い文字列、password、secret、connection string、GitHub token風文字列、AWS access key風文字列、Azure DevOps URLなどを対象とした簡易マスキング機能があります。

このマスキングは安全装置であり、すべての機密情報の検出や削除を保証するものではありません。AIツールや第三者へ送信する前に、出力内容を必ず確認してください。

第三者サービス

LogSlim拡張機能は、第三者サービスと通信しません。

このプライバシーポリシーページはGitHub上でホストされています。このページの閲覧者について、GitHubがGitHub自身のポリシーに基づいてアクセス情報を処理する場合があります。これはLogSlim拡張機能には影響しません。LogSlim拡張機能はGitHubへ通信しません。

変更

LogSlimのデータや権限の扱いが変わる場合、このポリシーを更新することがあります。更新時には「最終更新日」を変更します。
